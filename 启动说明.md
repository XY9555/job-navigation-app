# 求职导航系统 - 启动指南

## 🚀 快速启动

### 1. 环境要求
- Node.js 16+ 
- npm 8+
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 2. 安装依赖
```bash
# 安装前端依赖
npm install

# 安装后端依赖
cd backend
npm install
```

### 3. 启动服务
```bash
# 终端1：启动后端服务
cd backend
node server.js

# 终端2：启动前端服务
npm run dev
```

### 4. 访问应用
- **前端地址**: http://localhost:8080
- **后端API**: http://localhost:3000

## 📱 主要功能
- 用户注册/登录
- 简历创建和管理
- AI简历评测
- 职位匹配分析
- 面试问题生成
